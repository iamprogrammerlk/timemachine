# ⚒️ Usage

```
git add .

GIT_AUTHOR_DATE='1987-10-24 1:00:00 +0530' GIT_COMMITTER_DATE='1987-10-24 1:00:00 +0530' git commit -m 'MESSAGE'

git push
```

> [!WARNING] The second command must be in a single line, breaking it won't work as intended.
